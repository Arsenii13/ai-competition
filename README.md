# 🧠 AI Voxel Engine Challenge

🚀 This project was built as part of an **AI vs AI coding challenge** to recreate a Minecraft-style game from scratch.

Multiple AI models were given the same task:

> Build a playable voxel-based survival experience, starting from a minimal foundation and expanding toward Minecraft-like gameplay.>
##View the results at https://github.com/Arsenii13/ai-competition/wiki/

---

## 🏆 Competition Context

This project was developed alongside solutions generated by different AI systems, including:

* GitHub Copilot
* Claude (Anthropic)
* Codex / GPT-based models

Each model approached the problem differently:

* Some focused on features (biomes, caves, systems)
* Others focused on performance or structure

👉 This repository represents **my AI-assisted solution**, focusing on:

* A working core gameplay loop
* Clean structure for expansion
* Playable browser experience

## 🎯 Core Features

* 🌍 Procedural terrain generation (plains-style world)
* 🎥 First-person camera with mouse control
* 🏃 Player movement with gravity and jumping
* ⛏️ Block breaking system
* 🧱 Block placement system
* 🎒 Basic inventory tracking
* ⚡ Runs directly in the browser

---

## ⚖️ Design Approach

Instead of trying to replicate every Minecraft feature immediately, this project focuses on:

1. **Strong core engine**
2. **Playable experience early**
3. **Expandable architecture**

This approach prioritizes:

* Stability over feature overload
* Simplicity over complexity
* Real gameplay over demos
